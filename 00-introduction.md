## Introduction

### Qu'est-ce qu'un token fongible ?

Un **token fongible** est un actif numérique interchangeable, comme une monnaie. Chaque unité a la même valeur et peut être échangée contre une autre unité identique.

**Exemples :**
- Sur Ethereum : USDC, LINK, UNI (tokens ERC-20)
- Sur Solana : USDC, RAY, BONK (tokens SPL)

### Pourquoi Solana ?

- **Rapidité** : ~400ms par transaction (vs 12 secondes sur Ethereum)
- **Coût** : ~0.00025$ par transaction (vs 5-50$ sur Ethereum)
- **Scalabilité** : Peut traiter 65,000 transactions/seconde
- **Écosystème 2025** : +$11.3 milliards en market cap de tokens Solana

### Token vs Token-2022

Solana propose deux programmes pour créer des tokens :

| Caractéristique | Token (Original) | Token-2022 (Extensions) |
|----------------|------------------|-------------------------|
| **Statut** | Stable, feature-complete | Moderne, extensible |
| **Fonctionnalités** | De base | Avancées (frais de transfert, métadonnées, hooks, etc.) |
| **Compatibilité** | 100% supporté | 100% compatible + 16 nouvelles extensions |
| **Utilisation 2025** | Projets legacy | **Nouveaux projets professionnels** ✅ |
| **Exemples** | Anciens tokens | BERN (Transfer Fee), USDP (Permanent Delegate), WNS NFTs |

**Dans cette formation, nous utiliserons Token-2022 avec ses extensions avancées.**
